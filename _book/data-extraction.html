A Meta-Analysis of Psychotherapy Studies Conducted in Naturalistic Settings

Chapter 5 Data Extraction

5.1 Effect Size Calculation

There was 233 studies forwarded for data extraction from the full-text screening stage. Our unified effect-size metric for meta-analysis was paired samples Cohen’s d, dividing the mean group differance by the PreSD and adjusting the sampling variance using the pre-post Pearon’s r. This is consistent with the approach advocated Minami and collegues (2008). As very few studies used this precise effect-size variant we aspired to calculate them ourselves using the data reported within manuscripts. Unfortunately a large number of studies did not report sufficicent information for us to do this.

The majority of studies reported all required information apart from* r (185 studies). In these situations we e-mailed authors to request that, if able, they provide us with Pearson’s r Information was subsequently provided for XXX studies. For studies that remained without r at this point we imputed the value using an empirically established estimate (0.6, reference).

For 35 studies there was sufficient information to manually calculate Cohen’s d (i.e. did not report the preM, postM, and preSD). In these circumstances, if the manuscript reported an alternative varition of Cohen’s d (e.g. [preM-postM]/preSD) then we used took this value. If no Cohen’s d was available then we contacted authors to request the missing information. If authors were unable to provide this information then we sought to esimate (conversion) the required statistics using alternative available data (e.g. median, range, standard error, ANOVA, regression). If manuscript detail was still insufficient to calculate Cohen’s d then the study was not included in the meta-analysis, was kept in the narrative synthesis.
